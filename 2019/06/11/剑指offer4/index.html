23. 输入一个整数数组，判断该数组是不是某二叉搜索树的后序遍历的结果。如果是则输出Yes,否则输出No。假设输入的数组的任意两个数字都互不相同。

思路：递归。二叉搜索树的特点是中序遍历为各个结点按照大小的顺序排序，因此他的左子树所有值小于根且右子树所有值大于根。依此来看，则他的后序遍历会有一个特点：根节点在最末尾一个元素且左右子树分界点条件是a < root < b。因此可以找到拐点判断左右子树，然后依次递归这个过程。

24. 输入一颗二叉树的跟节点和一个整数，打印出二叉树中结点值的和为输入整数的所有路径。路径定义为从树的根结点开始往下一直到叶结点所经过的结点形成一条路径。(注意: 在返回值的list中，数组长度大的数组靠前)

思路：用栈递归实现所有路径的选择。这里可以选择两个栈，其中一个保存返回的list，另一个保存需要遍历的树。在这里可以定义一个新的函数实现递归判断，首先，可以设置一个标志位来判断某一节点是否存在左右子树，如果不存在，则返回该树，添加到返回list中；存在的话，则将该节点添加到第二个栈保存他的左右子树。最终返回该list即可。
